Feature 1: Move Between Rooms

Trigger: Player selects “move”
The player moves to the next room in the dungeon. The game updates the current room index and displays the new room’s description. If the room contains an enemy, combat may begin.

Feature 2: Combat System

Trigger: Player encounters an enemy
The player and enemy take turns attacking each other. Health is reduced based on attack power. Combat ends when either the player or enemy reaches zero health.

Feature 3: Room Exploration

Trigger: Player enters a room
Displays the room description. Indicates whether an enemy is present or if the room is the exit. May trigger events like combat or victory.

Feature 4: Player Health System

Trigger: During combat or events
Tracks player health throughout the game. If health reaches zero, the game ends. Health may be reduced by enemies or increased by healing events.

Feature 5: Escape Condition

Trigger: Player reaches exit room
If the player reaches the exit room while alive, the game ends with a victory message.
